## Overview
Having started to cook, you wanted to keep track of all your recipes. As a developer, your first thought was to build a quick app.

The first part is to be able to list down all the recipes and its
ingredients. 

## Specifications
1. Create a project called **recipebook**.
2. Inside the **recipebook** project, create an app called ledger.
3. The ledger app should have 3 accessible pages:
    - `/recipes/list`: List 2 recipes that link to each individual recipe
    - `/recipe/1`: List the ingredients for Recipe 1
    - `/recipe/2`: List the ingredients for Recipe 2
4. [This file](https://drive.google.com/file/d/1QnYz7wySUpliGoO4wc3w0lyD61SiU1g0/view?usp=drive_link) contains the necessary context for `/recipes/list`
5. [This file](https://drive.google.com/file/d/12XCKZVkO1iEajv5EO2eAx1-iGILKDkNu/view?usp=drive_link) contains the necessary context for `/recipe/1`
6. [This file](https://drive.google.com/file/d/1_UfHOsBcyxjvhOXWjIaPfRBrTaLzqpTW/view?usp=drive_link) contains the necessary context for `/recipe/2`
7. The pages need to have a header that contains the text "Recipe Book".
